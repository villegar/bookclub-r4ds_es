Datos ordenados

En (Grolemund and Wickham 2019, Cap 12) aprenders a:

  • Una forma consistente de ordenar datos

  • Herramientas del paquete tidyr:

    • https://tidyr.tidyverse.org/

  • Definicin de datos ordenados (Wickham 2014)

    • Cada variable forma una columa
    • Cada observacin forma una fila
    • Cada valor debe estar en su propia celda
    • Cada tipo de unidad de observacin forma una tabla

Valores faltantes

  • Explcitos: sealados con NA

  • Implcitos: no se encuentran sealados en los datos

    • En la fila 4 respecto a la variable return existe un valor faltante explcito

    • El dato del ao 2016 para el primer trimestre es un valor faltante implcito

Datos no-ordenados

  • Por qu utilizarlos?

    • Pueden tener ventajas sustanciales de rendimiento o de espacio
    • Diferentes campos han desarrollado sus propias convenciones para almacenar datos que pueden ser bastante diferentes a las convenciones de los datos ordenados
